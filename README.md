# Prato Certo - React (Vite + Tailwind + CSS)

Conteúdo do ZIP:
- Projeto Vite + React pronto para rodar.
- Tanto CSS tradicional (src/styles/styles.css) quanto suporte a Tailwind (src/styles/tailwind.css) foram incluídos.
- Para ver o Tailwind funcionando é necessário instalar dependências: `npm install` ou `yarn`.

Como usar:
1. `npm install`
2. `npm run dev`
3. Abra `http://localhost:5173`

Observações:
- Este scaffold inclui a lógica convertida do seu script.js para React (hooks + localStorage).
- Os estilos originais foram preservados em `src/styles/styles.css`. Também há `tailwind.css` com diretivas para uso do Tailwind.
